# หุ่นยนต์ซูโม — 4 โหมด 🤖
### LEGO SPIKE Prime — สร้างจากบล็อก (อย่าคัดลอกอย่างเดียว)

**เป้าหมาย:** ผลักคู่ต่อสู้ออกนอกวงกลม โดยที่ตัวเองไม่ตกลงไปเอง  
**กฎสำคัญ:** หุ่นยนต์ต้องรอ **5 วินาที** หลังจากเริ่มก่อนจะเคลื่อนที่ได้

หุ่นยนต์มีทุกอย่างพร้อมแล้ว:
| อุปกรณ์ | หน้าที่ |
|---|---|
| มอเตอร์ **A + E** | เคลื่อนที่ |
| เซ็นเซอร์สี **F + D** | เซ็นเซอร์ตรวจจับ **ขอบ** (เส้นขาว) |
| เซ็นเซอร์วัดระยะ **B** | เซ็นเซอร์ตรวจจับ **คู่ต่อสู้** |
| เซ็นเซอร์แรง **C** | **กันชน** (รู้ว่ากำลังผลักอยู่) |

---

## 🧱 โครงสร้างพื้นฐาน (ใช้ร่วมกันทุกโหมด)

> นักเรียนสร้างส่วนนี้ **ครั้งเดียว** แล้วเปลี่ยนแค่ "สมอง" (ลูป `forever`)

```
เมื่อกดปุ่ม [ขวา]
  ตั้งมอเตอร์เคลื่อนที่เป็น  A+E
  ตั้งความเร่งเป็น  เร็ว
  รอ 5 วินาที              <-- กฎซูโม (บังคับ!)
  วนซ้ำตลอด
      ┌── สมอง (เปลี่ยนตามโหมดด้านล่าง)
      └──
```

**กฎทองในทุกโหมด:** `if` แรกสุดในลูป `forever` ต้องตรวจ **ขอบ**  
(เซ็นเซอร์ F หรือ D เห็นสีขาว) → ถอยหลัง + หมุน
> 👉 คำใบ้สำหรับนักเรียน: *"ทำไมต้องตรวจขอบก่อน ไม่ใช่ตรวจทีหลัง?"*  
> (คำตอบที่ต้องค้นหา: ถ้าโจมตีก่อน จะตกก่อนที่จะได้ตรวจพื้น)

รีเฟลกซ์กันตก ("back off") ที่ต้องใส่ทุกโหมด:
```
เคลื่อนที่  ↓ (ถอยหลัง)  เร็ว , ช่วงสั้น ๆ
แล้วหมุน  (ขวา)  ช่วงสั้น ๆ   <-- กลับไปตรงกลาง
```

---

## 🛡️ โหมด 1 — SURVIVOR (ปลอดภัย)
*"ฉันชนะเพราะฉันไม่ตกเลย"* — ชนะหุ่นที่บุกแหลกแล้วทำตัวเองตก

```
วนซ้ำตลอด
  ถ้า  < F หรือ D เห็นสีขาว (ขอบ) >  แล้ว
      ถอยหลัง + หมุน            # ลำดับความสำคัญสูงสุด
  ถ้าไม่ ถ้า  < B ใกล้กว่า 36 ซม. >  แล้ว
      เริ่มเคลื่อนที่ตรง  ด้วยความเร็ว CRUISE (~65%)   # ผลัก แต่ควบคุมได้
  ถ้าไม่
      เริ่มหมุนช้า ๆ               # ค้นหาคู่ต่อสู้
```

---

## ⚡ โหมด 2 — BULLDOZER (บุกแหลก)
*"ความเร็วสูงสุด ลอดใต้คู่ต่อสู้ ผลักก่อนที่เขาจะตอบสนอง"* — ชนะหุ่นหนัก/ช้า

```
เมื่อเริ่ม (หลัง 5 วิ) :
  เคลื่อนที่ตรงด้วยความเร็วเต็มที่ 0.5 วินาที     # การพุ่งออกตัว: 0.5 วิแรกตัดสิน

วนซ้ำตลอด
  ถ้า  < F หรือ D เห็นสีขาว (ขอบ) >  แล้ว
      ถอยหลัง + หมุน            # ตาข่ายนิรภัย ตอบสนองเร็วมาก
  ถ้าไม่ ถ้า  < B ใกล้กว่า 36 ซม. >  หรือ  < C ถูกกด >  แล้ว
      เริ่มเคลื่อนที่ตรงด้วยความเร็วเต็มที่          # เต็มที่ตลอด
  ถ้าไม่
      เริ่มหมุนเร็ว               # ค้นหาเร็ว
```
> ⚠️ คำใบ้: *"Bulldozer อาจตกเองได้ถ้าพลาด จะปรับค่าขอบยังไงให้เบรกทัน  
> แต่ไม่เบรกเร็วเกินไป?"*

---

## 🥋 โหมด 3 — JUDOKA (สร้างสรรค์)
*"ไม่สู้หน้าตรง หลบแล้วโจมตีด้านข้าง"* — ชนะหุ่นที่แข็งแกร่งกว่าด้วยตำแหน่ง

```
วนซ้ำตลอด
  ถ้า  < F หรือ D เห็นสีขาว (ขอบ) >  แล้ว
      ถอยหลัง + หมุน
  ถ้าไม่ ถ้า  < B ใกล้กว่า 36 ซม. >  แล้ว
      หมุน  (ขวา)  ช่วงสั้น ๆ        # 1) เอียงมุม (หลบ)
      แล้วเคลื่อนที่ตรงด้วยความเร็วเต็มที่ 0.4 วินาที   # 2) พุ่งชนด้านข้าง
  ถ้าไม่
      เริ่มหมุนช้า ๆ           # ค้นหา
```
> ทำไมถึงได้ผล: หุ่นที่ถูกผลัก **ด้านข้าง** จะไม่มีแรงยึดเกาะต้านทาน  
> 👉 คำใบ้: *"มุมของการหมุนคือความลับทั้งหมด น้อยเกินไป = ผลักหน้าตรง (แพ้)  
> มากเกินไป = พลาด ค้นหาค่าที่เหมาะสมด้วยการทดสอบ"*

---

## 🦎 โหมด 4 — METAMORPH (ปรับตัว)
*"ฉันไม่ได้เดาคู่ต่อสู้ ฉันตอบสนองต่อสัญญาณที่รู้สึกได้ตอนนี้"*

**กรอบความจริง:** ด้วยเซ็นเซอร์วัดระยะ + กันชนเท่านั้น หุ่นยนต์ไม่สามารถ  
"จำแนกประเภทคู่ต่อสู้ที่อยู่ตรงหน้า" ได้ แต่มันสามารถ **รู้สึกว่ากำลังเกิดอะไรขึ้น** :
- แตะและเข็งไปข้างหน้าได้? → โหมด Bulldozer
- แตะแล้วติดค้างนาน? → โหมด Judoka (ถอยออกมา + โจมตีด้านข้าง)
- อยู่ที่ขอบ? → โหมด Survivor (ลำดับความสำคัญสูงสุด)
- เห็นเป้าหมาย? → พุ่งชน
- ไม่เห็นอะไร? → ค้นหา

```
วนซ้ำตลอด
  ถ้า  < F หรือ D เห็นสีขาว (ขอบ) >  แล้ว
      ถอยหลัง + หมุน                        # SURVIVOR: ไม่ตก
  ถ้าไม่ ถ้า  < C ถูกกด >  แล้ว
      ถ้า  < การสัมผัสกินเวลานาน >  แล้ว
          หมุนหนี + หมุนเอียง + พุ่งด้านข้าง  # JUDOKA: หลบเข้าข้าง
      ถ้าไม่
          เคลื่อนที่ตรงด้วยความเร็วเต็มที่    # BULLDOZER: ผลักแรง
  ถ้าไม่ ถ้า  < B ใกล้กว่า 36 ซม. >  แล้ว
      เริ่มเคลื่อนที่ตรงด้วยความเร็วเต็มที่   # เห็นเป้าหมาย -> พุ่ง
  ถ้าไม่
      เริ่มหมุนช้า ๆ                        # ค้นหา
```

> ⚠️ คำใบ้: *"สามเหลี่ยม Survivor/Bulldozer/Judoka (เป่ายิงฉุบ) เป็นเรื่องสอนน่ารัก  
> แต่ไม่ใช่กฎทางฟิสิกส์ Metamorph ไม่ได้สัญญาว่าจะชนะ: มันแค่พยายามไม่ติดอยู่กับกลยุทธ์ที่ล้มเหลวอยู่"*

## 🎛️ ตัวเลขที่นักเรียนต้องหาด้วยตัวเอง
*(นี่คือการเรียนรู้ — เราให้โครงสร้าง เขาหาตัวเลขเอง)*

| การตั้งค่า | ผลกระทบ | วิธีหา |
|---|---|---|
| **ค่าระยะ B** (เช่น 36 ซม.) | ไกลเกิน → พุ่งเข้าอากาศ ; ใกล้เกิน → ตอบช้า | ทดสอบกับหุ่นที่อยู่นิ่ง |
| **ความเร็ว** (%) | การแลกเปลี่ยนระหว่างแรงผลัก vs การควบคุม | เพิ่มจนลื่น แล้วลดลงหนึ่งขั้น |
| **เวลาถอยหลัง** | สั้นเกิน → ตกซ้ำ ; นานเกิน → เสียเวลา | สังเกตบนสนาม |
| **มุมการหมุน** (Judoka) | พลาด vs โจมตีหน้าตรง | การปรับละเอียดที่สุด |
| **เวลา stalemate** (Metamorph) | สั้นเกิน → ตื่นตระหนก ; นานเกิน → ติดค้าง | เมื่อการผลักถูกขวาง |

**กฎการเรียนรู้:** เปลี่ยน **ทีละหนึ่ง** ตัวเลข แล้วลองแข่ง แล้วสังเกต  
ถ้าเปลี่ยนหลายอย่างพร้อมกัน จะไม่เข้าใจอะไรเลย

## 🧪 วิธีทดสอบ (การแข่งขันภายใน)
1. สร้าง **Survivor** ก่อน (รากฐาน: กันตก)
2. คัดลอก 2 ครั้ง → ทำเป็น **Bulldozer** และ **Judoka**
3. ให้ทั้งสามต่อสู้กันเอง → นักเรียนจะเห็นว่าสไตล์ไหนชนะและทำไม
4. เมื่อทั้งสามใช้งานได้ ลอง **Metamorph** — แต่บอกความจริง: มันตอบสนอง ไม่ใช่เดา
5. เอาตัวที่ชนะไปแข่งจริง และต้อง **อธิบายให้กรรมการฟังได้**
